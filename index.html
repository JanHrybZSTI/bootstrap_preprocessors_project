Co to jest preprocesor w CSS?

Preprocesor CSS to program, który umożliwia generowanie CSS na podstawie własnej, unikalnej składni preprocesora. Istnieje wiele preprocesorów CSS do wyboru, jednak większość preprocesorów CSS dodaje pewne funkcje, które nie istnieją w czystym CSS, takie jak mieszanie, selektor zagnieżdżania, selektor dziedziczenia i tak dalej. Te funkcje sprawiają, że struktura CSS jest bardziej czytelna i łatwiejsza w utrzymaniu. W tym artykule postaram się opisać trzy preprocesory, czyli Sass'a LESS'a oraz Stylus'a

Sass

Sass to preprocesor, który pozwala na włączenie do CSS takich rzeczy, jak zmienne, zagnieżdżanie, miksowanie, dziedziczenie itp. Jest to nadzbiór css, więc każdy kod css jest również prawidłowy. Obsługuje dwa rodzaje składni, scss i sass. SCSS jest nieco bardziej rygorystyczny, ponieważ zawiera nawiasy klamrowe i średnik, podczas gdy SASS jest silnie oparty na odpowiednich wcięciach.

LESS

Less to preprocesor CSS, który ułatwia dostosowywanie, zarządzanie i ponowne używanie arkuszy stylów na stronie internetowej. Less jest rozszerzeniem CSS i dynamicznym językiem arkuszy stylów, który można uruchomić po stronie klienta lub serwera.

Stylus

Stylus to dynamiczny język preprocesora arkuszy stylów, który jest kompilowany w kaskadowych arkuszach stylów (CSS). Został zaprojektowany na podstawie Sass'a i LESS'a. Jest uważany za czwartą najczęściej używaną składnię preprocesora CSS. Został stworzony przez TJ Holowaychuka, byłego programistę Node.js i twórcę języka Luna. Jest napisany w JADE i Node.js.
