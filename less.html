LESS

O preprocesorze

Less to preprocesor CSS, który ułatwia dostosowywanie, zarządzanie i ponowne używanie arkuszy stylów na stronie internetowej. Less jest rozszerzeniem CSS i dynamicznym językiem arkuszy stylów, który można uruchomić po stronie klienta lub serwera.

Zmienne

Są to dość oczywiste:

@width: 10px;
@height: @width + 10px;
    
#header {
    width: @width;
    height: @height;
}

Zagnieżdżanie

Less daje możliwość użycia zagnieżdżania zamiast lub w połączeniu z kaskadowaniem.

#header {
    color: black;
    .navigation {
        font-size: 12px;
    }
    .logo {
        width: 300px;
    }
}

Mixiny

Domieszki to sposób na włączenie („wmieszanie”) zestawu właściwości z jednego zestawu reguł do innego zestawu reguł.

.bordered {
    border-top: dotted 1px black;
}
                
.post a {
    color: red;
    .bordered();
}

Funkcje

Less zapewnia różnorodne funkcje, które przekształcają kolory, manipulują łańcuchami i wykonują obliczenia. Są one w pełni udokumentowane w opisie funkcji. Korzystanie z nich jest dość proste. Poniższy przykład używa wartości procentowej do konwersji 0,5 na 50%, zwiększa nasycenie koloru podstawowego o 5%, a następnie ustawia kolor tła na taki, który jest rozjaśniony o 25% i obrócony o 8 stopni:

@base: #f04615;
@width: 0.5;
                
.class {
    width: percentage(@width); // returns `50%`
    color: saturate(@base, 5%);
}
